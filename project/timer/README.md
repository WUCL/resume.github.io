Timer 計時器

#### 使用方法
1. 點擊螢幕可執行 Start / Pause
2. 可選擇 TABATA 4分鐘 / 20s:30 10分鐘
3. 可於暫停時 Stop

#### 聲音
1. 倒數三秒會出音效
2. run / break 切換時會出音效
3. 結束時會有音效

#### 字體
1. [Oswald](https://fonts.google.com/specimen/Oswald)

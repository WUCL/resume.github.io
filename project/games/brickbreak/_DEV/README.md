# BrickBreak
###### 參考製作
[2D breakout game using pure JavaScript](https://developer.mozilla.org/zh-CN/docs/Games/Tutorials/2D_Breakout_game_pure_JavaScript)

----
一個古老遊戲，打磚塊。

##### 遊戲玩法
以 [左右鍵] 控制 [接板]
以 [Space] 開啟遊戲 (發球)
##### 聲音
1. 音量總開關 on/off
2. [球] 撞擊 [接板]/[磚] 出音效
3. 背景音樂
